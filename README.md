# Bicol-Dialect-Part-of-Speech-Tagger-BPOS
BPOS: A PART-OF-SPEECH TAGGER USING  CONDITIONAL RANDOM FIELD  FOR BICOL DIALECT

## Instructions

1. Install:

```
pip install -r requirements.txt
```

2. Inside bpostagger: ``` python BPOS_Tagger.py ```
